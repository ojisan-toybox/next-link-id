# next-link-id

id jump on Next

```sh
npx next dev
```

| ページ                                                                        | 成否                 |
| ----------------------------------------------------------------------------- | -------------------- |
| Link で#つけて遷移                                                            | ページ内リンクに成功 |
| router.push で#つけて遷移                                                     | ページ内リンクに成功 |
| Link で#つけて遷移(ただし遷移後のカードは API 取得)                           | ページ内リンクに失敗 |
| Link で#つけて遷移(ただし遷移後のカードは API 取得/ref 使って scrollIntoView) | ページ内リンクに成功 |
